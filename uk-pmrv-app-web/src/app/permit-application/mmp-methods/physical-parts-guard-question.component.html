In line with Annex VI section 2(c) of the FAR, you must list all physical parts of installation and units which serve more than one sub-installation, for example heat supply systems, jointly used boilers and CHP units.

Units which only serve one sub-installation should not be listed here. Describe these units in the sub-installations section.

If a boiler produces measurable heat that is consumed by two product benchmark sub-installations, the boiler should be listed and both sub-installations selected from the drop-down list. If the heat is consumed by only one of the two sub-installations, add this in the sub-installations section.
