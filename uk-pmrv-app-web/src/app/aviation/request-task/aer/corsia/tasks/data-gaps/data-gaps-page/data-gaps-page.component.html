A data gap is where the CO2 emissions from a flight cannot be determined from primary data sources or reconstructed from secondary data sources.

The number of flights where data gaps occurred should not exceed 5% of the number of flights with offsetting obligations over the monitoring year.

If you exceed the 5% threshold, you must tell your regulator what remedial action you are taking to address the frequency of the data gaps. This must be done outside of the reporting process or as a separate communication to your regulator.

If you do not exceed the 5% threshold and are using a fuel use monitoring method, you need to account for the data gaps using the ICAO CORSIA CO2 estimation and reporting tool (CERT) .
