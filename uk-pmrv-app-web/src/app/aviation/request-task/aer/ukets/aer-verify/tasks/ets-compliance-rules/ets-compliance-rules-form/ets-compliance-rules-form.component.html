Have the emissions monitoring plan requirements and conditions been met?

You must assess whether the aircraft operator has met the requirements of the approved emissions monitoring plan, especially relating to the emissions report.

Have the requirements of the UK ETS Order, which amends the MRR, been met?

You must assess if the aircraft operator has met the requirements of the UK ETS Order .

Can you verify the detail and source of data?

You should carry out detailed testing which at minimum covers:

  • analytical procedures
  • data verification
  • checking the monitoring approach
  • checking the data flow activities and the systems used, including IT systems

As part of this data verification, whilst checking the aircraft operator’s emissions monitoring plan and procedures, you’ll need to:

  • trace the data back to the primary data source
  • cross check data with external data sources
  • perform reconciliations
  • check thresholds for appropriate data
  • carry out recalculations

Were control activities documented, implemented, maintained and effective to reduce any risks?

You should carry out detailed testing which at minimum covers:

  • analytical procedures
  • data verification
  • checking the monitoring approach

You should also check that the control activities are well documented, implemented and maintained, and are effectively reducing any inherent risks.

Were procedures in the emissions monitoring plan documented, implemented, maintained and effective to reduce risks?

To ensure that the emissions monitoring plan is implemented correctly, you need to check that the procedures in it are effectively reducing the inherent risks and control risks.

The procedures must be:

  • implemented correctly
  • documented correctly
  • maintained properly

Has data verification been completed as required?

You must test the data in detail. This means you have to:

  • trace all data back to its primary source
  • cross-check data with external sources
  • make reconciliations and recalculations
  • check thresholds for appropriate data

As part of this verification you’ll have to check:

  • consistency of the aggregated reported data with primary source data
  • any measured values if the aircraft operator uses a measurement-based approach
  • reliability and accuracy of the data

Has the monitoring approach been applied correctly?

You must check that the monitoring approach and procedures set out in the approved emissions monitoring plan have been correctly applied and implemented.

Where the aircraft operator has deviated, this should be reported as a non-conformity or recommended improvement.

Has the completeness of flights or data been compared with air traffic data?

You must check the route data against that submitted to the aircraft operator by ATC providers, for example route charges invoices, report data obtained from Eurocontrol.

Have consistency checks been made between reported data and 'mass & balance' documentation?

Have consistency checks been made between aggregate fuel consumption and fuel purchase or supply data?

Were methods used for applying missing data appropriate?

You must check that:

  • the methods used for completing any missing data in the monitoring plan were appropriate for the specific situation, and that the operator has applied them correctly
  • the control activities which the operator has used to prevent missing data are effective

If the aircraft operator has approval to use other methods, you must check that they have applied and documented them correctly.

If they do not have approval to apply the data gap method used, you must check that their approach to addressing the gaps does not underestimate their emissions, and lead to material misstatements.

Has the regulator guidance on monitoring and reporting been met?

You must apply the harmonised standards which meet the essential requirements of the EC regulations for GHG accreditation and verification bodies.

You’ll also need to apply processes and arrangements for:

  • communicating with the aircraft operator and other parties
  • protecting the confidentiality of information you obtain
  • dealing with approvals
  • issuing a revised verification report after you have issued it to the operator

Have any non-conformities from last year been corrected?

You need to assess if the operator has corrected non-conformities from the verification report relating to the previous monitoring period.

If they have not, will this increase the risk of misstatements?
