Using substitute data to estimate fuel consumption

Describe the surrogate method you will use to estimate fuel consumption where data is missing.

If primary or secondary data is missing, you must substitute it with data calculated using a surrogate method that your regulator approves (for example, a conservative estimate of fuel consumption based on historic averages for the same route).

If substitute data cannot be calculated using this method then you can use the Eurocontrol Small Emitters Tool to calculate your emissions.
