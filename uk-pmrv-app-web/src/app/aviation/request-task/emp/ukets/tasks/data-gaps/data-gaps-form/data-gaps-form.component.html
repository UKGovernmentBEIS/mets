Data gaps when monitoring fuel consumption

Describe the systems and procedures for identifying data gaps and continually assessing whether the 5% data gap flight threshold has been reached.

The number of flights where data gaps occurred should not exceed 5% of the total reported flights over the monitoring year.

If you exceed the threshold, you must:
  • tell your regulator what remedial action you are taking to address the frequency of data gaps
  • state the percentage of flights with data gaps in your annual emissions report, and provide an explanation

The procedure should show how, on an ongoing basis, you:

  • monitor for data gaps, so you can define these when primary and secondary data is missing
  • monitor the number against the threshold, and inform your regulator if necessary
