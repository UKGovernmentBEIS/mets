Fuel uplift
Measurement of fuel uplift and block hours per flight

Describe how you monitor fuel consumption and block-hour per flight in owned and leased-in aircraft.

You should include:

  • the exact points in time for the measurement of block hours per flight (necessary to calculate the fuel consumption per flight for international flights with zero uplift, and for the following flights)
  • a description of the measurement equipment
  • the procedures for recording, retrieving, transmitting and storing information
  • reference to any relevant documentation
  • the exact points in time for the measurement of block hours per flight
  • a description of the measurement equipment
  • the procedures for recording, retrieving, transmitting and storing information
  • reference to any corresponding documentation
FN = fuel consumed for the flight under consideration (flight = N) determined using fuel uplift (in tonnes)
UN = fuel uplift for the flight considered, measured in volume and multiplied with a density value (in tonnes)

For flights without a fuel uplift, you should use the following formula to allocate fuel use from the prior fuel uplift proportionally to block-hour (BHN):

FN = UN * [BHN / ( BHN + BHN+1 + ...+ BHN+n)]
FN+1 = UN * [BHN+1 / (BHN + BHN+1 + ...+ BHN+n)]
FN+n = UN * [BHN+n / (BHN + BHN+1 + ...+ BHN+n)]
