You should carry out detailed testing which at minimum covers:

  • analytical procedures
  • data verification
  • checking the monitoring approach
  • checking the data flow activities and the systems used, including IT systems

As part of this data verification, whilst checking the operator’s monitoring plan and procedures, you’ll need to:

  • trace the data back to the primary data source
  • cross check data with external data sources
  • perform reconciliations
  • check thresholds for appropriate data
  • carry out recalculations
