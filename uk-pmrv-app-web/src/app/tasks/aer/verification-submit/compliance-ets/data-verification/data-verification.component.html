You must test the data in detail. This means you have to:

  • trace all data back to its primary source
  • cross-check data with external sources
  • make reconciliations and recalculations
  • check thresholds for appropriate data

As part of this verification, for the monitoring plan or monitoring approach plan, you’ll have to check:

  • the boundaries of the installation (for the emissions report)
  • consistency of the aggregated reported data with primary source data
  • any measured values if the operator uses a measurement-based approach
  • reliability and accuracy of the data

If you’re verifying a baseline data, new entrant data or annual activity level report, you’ll also have to check:

  • the boundaries of the installation and its sub-installations
  • completeness of source streams and emission sources
